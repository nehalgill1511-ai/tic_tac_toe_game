# tic_tac_toe_game
Tic Tac Toe Game

This is a simple Tic Tac Toe game developed using Python. It is a two-player game where players take turns using X and O on a 3x3 grid. The goal is to align three matching symbols in a row, column, or diagonal to win.

Features

- Two-player gameplay
- Win and draw detection
- Simple interface
- Easy to run

Technologies Used

- Python
- Tkinter / Python IDLE

Learning Outcome

This project helped me understand Python programming, game logic, problem-solving, and basic Object-Oriented Programming concepts.

Challenges Faced

- Creating game logic
- Detecting winners
- Managing player turns
- Debugging errors
